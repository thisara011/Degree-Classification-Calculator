# 🎓 Degree Classification Calculator

A simple, interactive web application that allows students to calculate their final degree classification based on individual subject credits and marks.

![{1A2CA36C-77CF-4B18-AEB7-5E8BC510490D}](https://github.com/user-attachments/assets/f782339f-8201-4fbf-b9e3-cbdf7aacbde5)


---

## 🚀 Features

- Add multiple subjects with credit value and mark
- Automatically calculates:
  - Weighted total score
  - Average percentage
  - Final classification (e.g., First Class, Second Upper)
- Visual color indicators for each classification
- Fully client-side (no backend required)
